# Helpdesk Automation Toolkit (PowerShell)

## Overview
A small set of PowerShell scripts designed to automate common Tier 1 / Tier 2 helpdesk tasks in an Active Directory environment.

## Scripts
- `reset-password.ps1` — resets AD user password and forces change at next logon
- `unlock-user.ps1` — unlocks a locked AD account
- `new-user.ps1` — creates a new AD user in a chosen OU and optionally adds group membership

## Skills Demonstrated
- PowerShell scripting for IT support
- Active Directory user lifecycle operations
- Standard helpdesk tasks performed efficiently and consistently
